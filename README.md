# CodeAlpha - Stock Portfolio Tracker

## 📌 Project Description
This is a Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Python Programming Internship.

## 🚀 Features
- Add stock names and quantities
- Uses predefined stock prices
- Calculates total investment value
- Displays portfolio summary
- Saves portfolio details to a text file

## 🛠 Technologies Used
- Python 3

## ▶️ How to Run
1. Clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run:
   ```
   python stock_portfolio_tracker.py
   ```
4. Enter stock names and quantities as prompted.

## 📂 Output
- Displays the total investment value.
- Saves the portfolio summary in `portfolio.txt`.
